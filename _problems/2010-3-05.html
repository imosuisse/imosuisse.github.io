Consider the vertices of a regular \(n\)-corner and connect them somehow with sides or diagonals to form a closed path that passes through each corner exactly once. A parallel pair is a set of two different parallel lines in this line. Show:

  1. If \(n\) is even, then there is always at least one parallel pair.

  2. If \(n\) is odd, then there is never exactly one parallel pair.
