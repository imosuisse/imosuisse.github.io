If \(p\geq 5\) is a prime number, let \(q\) denote the smallest prime number such that \(q>p\) and let \(n\) be the number of positive divisors of \(p+q\) (\(1\) and \(p+q\) included).

  1. Prove that no matter the choice of \(p\), the number \(n\) is always at least \(4\).

  2. Find the actual minimal value \(m\) that \(n\) can reach among all possible choices for \(p\). That is:

    • Give an example of a prime number \(p\) for which the value \(m\) is reached.

    • Prove that there is no prime number \(p\) for which \(n\) is smaller than \(m\).
