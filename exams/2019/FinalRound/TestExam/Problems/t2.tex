\de{Sei $S$ eine Teilmenge von $\left[0,1\right]$ bestehend aus 2019 disjunkten Intervallen $\left[a_1, b_1\right], \ldots , \left[a_{2019},b_{2019}\right]$. Nehme an, dass es für jedes $d \in \left[0,1\right]$ Zahlen $x,y \in S$ gibt, sodass $\abs{x - y} = d$ gilt.
Bestimme den kleinstmöglichen Wert, den der Ausdruck 
\[
    \sum\limits_{k=1}^{2019} (b_k - a_k)
\]
annehmen kann.

\textit{Bemerkung: Ein Intervall $\left[a,b\right]$ ist die Menge der reellen Zahlen $x$ mit $a \leq x \leq b$.  }}
\fr{Soit $S$ un sous-ensemble de $\left[0,1\right]$ composé de 2019 intervalles disjoints $\left[a_1, b_1\right], \ldots , \left[a_{2019},b_{2019}\right]$. On suppose que pour tout $d \in \left[0,1\right]$ il existe $x,y\in S$ tels que $\lvert x - y \rvert = d$.
Déterminer la valeur minimale que peut atteindre l'expression
\[
    \sum\limits_{k=1}^{2019} (b_k - a_k).
\]

\textit{Remarque: un intervalle $\left[a,b\right]$ est l'ensemble des nombres réels $x$ avec $a \leq x \leq b$.}}
\en{Let $S$ be a subset of $\left[0,1\right]$ consisting of 2019 disjoint intervals $\left[a_1, b_1\right], \ldots ,\left[a_{2019},b_{2019}\right]$. Assume that for all $d \in \left[0,1\right]$ there exist $x, y\in S$ such that $\lvert x - y \rvert = d$ holds.
Determine the smallest value that the expression
\[
    \sum\limits_{k=1}^{2019} (b_k - a_k)
\]
can attain.

\textit{Remark: An interval $\left[a,b\right]$ is the set of all real numbers $x$ such that $a \leq x \leq b$.}
