\en {
For which integers $n\geq 2$ can we arrange the numbers $1,2,\dots,n$ in a row, such that for all integers $1\leq k\leq n$ the sum of the first $k$ numbers in the row is divisible by $k$?
}

\de {
Für welche ganzen Zahlen $n\geq 2$ können wir die Zahlen $1,2,\dots,n$ in einer Reihe anordnen, sodass für alle ganzen Zahlen $1\leq k\leq n$ die Summe der ersten $k$ Zahlen in der Reihe durch $k$ teilbar ist?
}
