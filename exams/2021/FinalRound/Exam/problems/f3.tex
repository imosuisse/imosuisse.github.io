\en {Find all finite sets $S$ of positive integers with at least two elements, such that if $m>n$ are two elements of $S$, then
$$\frac{n^2}{m-n}$$
is also an element of $S$.

}

$$\frac{n^2}{m-n}$$
auch ein Element von $S$.
}
