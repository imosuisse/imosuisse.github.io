\fr{Soient $a,b,c$ les longueurs des côtés d'un triangle. Montrer que
\[
\sqrt{6} \sqrt{a+b+c} \le \displaystyle \sum_{cyc} \frac{a+b}{\sqrt{a+c}} < 2 \sqrt{2(a+b+c)}.
\]}
\en{
Let $a,b,c$ be the lengths of the sides of a triangle. Prove that
\[
\sqrt{6} \sqrt{a+b+c} \le \displaystyle \sum_{cyc} \frac{a+b}{\sqrt{a+c}} < 2 \sqrt{2(a+b+c)}.
\]
}

\de{Seien $a,b,c$ die Längen der Seiten eines Dreiecks. Zeige, dass gilt:

\[
\sqrt{6} \sqrt{a+b+c} \le \displaystyle \sum_{cyc} \frac{a+b}{\sqrt{a+c}} < 2 \sqrt{2(a+b+c)}.
\]
}
